# ✈️ Planedoro

![planedoro](https://github.com/user-attachments/assets/5a3195fc-3d25-4524-9da0-0d8262c76634)

> 🧘 **Planedoro** is a minimalist, Pomodoro-style focus timer set in a calming airplane cabin environment. Built with Vite, Vue 3, TypeScript, and Tailwind CSS, it offers a distraction-free workflow, rigid 50/10 focus cycles, and a responsive PWA experience.

---

## 🟢 Live Demo

Access the application:  
[faleonfall.github.io/Planedoro](https://faleonfall.github.io/Planedoro/)

---

## ✨ Features

- **Airplane Aesthetic:** Immersive cabin interface designed for deep focus
- **Rigid Cycles:** Pre-configured 50-minute work / 10-minute break intervals
- **Distraction-Free:** Minimalist layout to keep you in the zone
- **Progressive Web App:** Works offline and can be added to the home screen
- **Responsive Design:** Optimized for desktop, tablet, and mobile screens
- **Intuitive Controls:** Simple keyboard shortcuts for workflow efficiency
- **Accessibility Focused:** Clear typography and high-contrast elements
- **Instant Performance:** No loading screens, immediate start

---

## 🛠️ Technology Stack

- [Vue 3](https://vuejs.org/)  
- [Vite](https://vitejs.dev/)  
- [TypeScript](https://www.typescriptlang.org/)  
- [Tailwind CSS](https://tailwindcss.com/)  
- PWA Support via Web Manifest

---

## 📲 Usage

- **Online Access:** [Planedoro Live](https://faleonfall.github.io/Planedoro/)  
- **Installation:** Use “Add to Home Screen” for an app-like experience  
- **Keyboard Shortcuts:**
    - `Spacebar` — Start / Pause Timer  
    - `R` — Reset Cycle  
- **Focus Mode:** Set the timer and let the cabin ambience help you work
